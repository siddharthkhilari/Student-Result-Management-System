# Student Result Management System

This is a simple console-based Python project.

The program takes student details and subject marks,
calculates total marks and percentage,
and assigns grade using conditional statements.

## Features
- Input student name and roll number
- Input marks of three subjects
- Calculate total and percentage
- Assign grade (A, B, C, Fail)
- Display final result

## Technology Used
- Python

## How to Run
1. Open terminal in project folder
2. Run the command:
   python student_result.py
